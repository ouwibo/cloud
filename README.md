# Ouwibo Portfolio Template

A clean standalone React/Vite portfolio landing page template.

## Features

- Sky/cloud background
- Transparent animated hamster logo
- Smooth glass loader
- Glass CTA cards
- About / Blog panel
- Credit: **Built by Ouwibo**
- No Zo branding/source included

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Edit

- Main React code: `src/main.tsx`
- Styles/animations: `src/styles.css`
- Assets: `public/`
- Favicon/logo: `public/site-logo-transparent.png`
